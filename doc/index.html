Name

snd2nes — converts a sound file to NES DMC format

Synopsis

snd2nes [ option ...] {file}

Description

snd2nes reads a sound file and converts it to the format supported by the NES DMC hardware.

Options

--output=file

Store the output in file. If this option is not specified, the output will be written to standard output.

--verbose

Print progress information to standard output.

--help

Give a help list.

--usage

Give a short usage message.

--version

Print program version.

Files

Output

The output is raw DMC data suitable for playback on the NES.

Examples

To convert mysound.wav:

snd2nes mysound.wav
